Html5 Tags:

Header

- This element specifies a header for a document or section.
It should be used as a container for introductory content.

Nav

- This element defines a set of navigation links.
It is intended for large blocks of navigation links.
However, not all links in a document should be inside a element!

Aside

- This element defines some content aside from the content it is placed in (like a sidebar).

Article

- This element specifies independent, self-contained content.

Section

- This element defines a section in a document.

Time

- Defines a date/time.

Footer

- This element specifies a footer for a document or section.

Table

- Defines a table.
